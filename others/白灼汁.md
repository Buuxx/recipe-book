# 白灼汁

## Ingredients

| Ingredient | Amount | 
|------------|--------|
| 洋葱 Onion | 1/8 头 |
| 姜 Ginger | 4 片 |
| 小葱 Green Onion | 4根 |
| 香菜根 Cilantro Root | 1-2 根 |
| 蒸鱼豉油 | 2 汤勺 |
| 生抽 Light Soy Sauce| 2 汤勺 |
| 蚝油 Oyster Sauce | 1 汤勺 |
||||||||


## Preparation

姜切片，洋葱切丝，小葱切段，晾干（防止炸的时候溅油）

## Cooking Steps

1. 锅里倒油，油量大概能铺满锅底，半炸半煎的效果。把准备好的洋葱、姜、小葱、香菜根都放进去炸
2. 炸到葱姜有些焦黄的时候，放入2勺生抽，2勺蒸鱼豉油，和1勺蚝油，搅拌均匀
3. 等到酱汁烧开，大量冒出泡泡后，可以关火。晾凉后过滤装盒

## Tips

### Storage

1. 放在干燥的密封罐里冷藏保存3-5天
2. 用冰盒冷冻，冻好后放到密封袋里，随用随取
